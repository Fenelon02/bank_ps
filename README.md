# Bendis Bank

- Uma aplicação web moderna e responsiva de um banco digital, construída com as tecnologias mais atuais.
- Para visualizar: https://news-page-red.vercel.app/

## Sobre o Projeto

**Bendis Bank** é uma plataforma bancária interativa que oferece uma experiência de usuário fluida e intuitiva. O projeto foi desenvolvido com foco em design moderno, performance e acessibilidade.

### Principais Características

- ✨ **Interface moderna** com animações suaves (GSAP)
- 🎨 **Design responsivo** otimizado para todos os dispositivos
- ⚡ **Performance otimizada** com Next.js 16
- 🔧 **TypeScript** para segurança de tipos
- 🎭 **Componentes reutilizáveis** com Radix UI
- 📱 **Carrossel interativo** com Embla Carousel
- 🌈 **Tailwind CSS** para estilização moderna
- 🎯 **Acessibilidade** garantida

## 🚀 Tecnologias Utilizadas

### Frontend
- **Next.js** - Framework React moderno
- **React** - Biblioteca UI
- **TypeScript**  - Linguagem com tipagem
- **Tailwind CSS**  - Framework de CSS utilitário
- **GSAP**- Animações avançadas
- **Shadcn/UI** - Componentes pré-prontos 
- **Lucide React** - Ícones SVG
- **Radix UI** - Componentes acessíveis

## 🔧 Instalação

### Pré-requisitos
- Node.js 18+ instalado
- npm ou yarn como gerenciador de pacotes

### Passos

1. **Clone o repositório**
```bash
git clone https://github.com/Fenelon02/bank_ps.git
cd bank_ps
```

2. **Instale as dependências**
```bash
npm install
```

3. **Inicie o servidor de desenvolvimento**
```bash
npm run dev
```

4. **Acesse a aplicação**
Abra seu navegador e visite `http://localhost:3000`

## 📝 Scripts Disponíveis

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia servidor de desenvolvimento |
| `npm run build` | Cria build para produção |

## 🎨 Seções da Aplicação

A aplicação é composta pelas seguintes seções principais:

1. **Header** - Navegação principal
2. **Hero Section** - Seção de destaque com chamada principal
3. **Benefits** - Benefícios do serviço
4. **Marquee** - Faixa de movimento contínuo
5. **Credit Card Section** - Apresentação de cartão de crédito
6. **Offered Services** - Serviços disponíveis
7. **API Card** - Informações sobre API
8. **FAQ Section** - Perguntas frequentes
9. **Footer** - Rodapé da página
